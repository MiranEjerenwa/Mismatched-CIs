Mismatched CIs dashboard provides users the ability to identify and analyze mismatched server CI owner and billing data, in an effort to eliminate misaligned billing charges.

Development and Deployment

Application Information



Story Board

Need to be able to log into app and manage CI data based on user credentials • Should be able to upload a CSV file containing CI, Owning and Billing Department data • Upon data import, filter out CI data that contains a match between Dept Owner and Dept Billed by querying DB by User/DeptID • Display only missmatched CI data as a list on HTML dashboard • Add funtionality to notify / email the owner of mis-alinged CI data to correct in the CMDB • • •

Technology

Mismatched CI Dashboard is a web application using a Node and Express Web Server, backed by a MySQL Database. Sequelize is being used as the interface between the two system (ORM). The application is deployed on a public cloud platform, Heroku, for the Production environment. User authorization and data access is achieved using JWT. ..something about Web app secutiry (owasp)...... To meet new technology requirements,...
